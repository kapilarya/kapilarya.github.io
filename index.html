About the Visualization

This project used data from American Community Survey (ACS) 2012-2016 data to visualize the employment characteristics of workers by gender. Two characteristics were analyzed:

  • Hours worked per week
  • Income by educational attainment

The visualization has following characteristics:

  • Interactive hybrid structure
  • Uses scenes to present appropriate information
  • Uses annotations to highlight key observations
  • Uses parameters (Page numbers)
  • Uses triggers (Radio buttons for gender)
